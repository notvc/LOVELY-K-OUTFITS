# Lovely K Outfits

A modern, responsive e-commerce boutique website designed with a focus on elegant typography and a premium user experience. 

## ✨ Features

*   **Dynamic Theming**: Integrated Dark and Light modes with smooth transitions using CSS custom properties.
    *   **Light Mode**: Features a warm Peach and Burgundy palette.
    *   **Dark Mode**: Features a sophisticated deep purple and coral aesthetic.
*   **Interactive Shopping Cart**: 
    *   Real-time item count badge.
    *   Dropdown mini-cart for quick viewing and quantity adjustments.
    *   Mobile-optimized fixed positioning.
*   **Responsive Product Grids**: 
    *   Multi-column layouts that adapt seamlessly from desktop (5 columns) to mobile (1 column).
    *   Skeleton shimmer loading states for a high-end performance feel.
    *   Hover effects including image scaling and "Double-click to browse" prompts.
*   **Custom Typography**: Utilizes `Playfair Display` for a luxury feel and `Montserrat` for readability, alongside custom-loaded boutique fonts.
*   **Advanced UI Components**:
    *   **Expanding Search Bar**: A space-saving animated search input.
    *   **Testimonial Carousel**: A smooth, interactive slider for customer feedback.
    *   **Scroll Reveal**: Entrance animations for sections as the user scrolls.
    *   **Preloader**: A custom-branded loading animation for professional site entry.
*   **Contact & Location**: A stylized contact form with character counters and an integrated map with a custom editorial grayscale filter.

## 🛠️ Technology Stack

*   **HTML5**: Semantic structure.
*   **CSS3**: 
    *   Custom Properties (Variables) for theming.
    *   Flexbox and CSS Grid for complex layouts.
    *   Clamp/Viewport units for fluid typography.
    *   Cubic-bezier transitions for high-quality animations.
*   **JavaScript (Implied)**: Theme switching, cart logic, carousel functionality, and scroll animations.
